使用命令行添加已存在的工程到GitHub

发表于 2016-04-12   |   分类于 技术   |  
一、在Github主页创建新的仓库

  1. 添加项目名
  2. 添加项目描述
  3. 创建忽略文件
  4. 创建声明文件
二、打开命令行，cd到等待提交的工程目录
三、初始化本地目录作为git本地仓库
git init
四、添加文件到本地仓库，暂存文件
git add .
五、提交暂存的文件
git commit -m "First commit"
六、添加别名
git remote add origin remote repository URL
添加别名
git remote -v
验证别名
七、合并远程仓库到本地仓库
git pull origin master
八、提交本地仓库的变更到远程仓库
git push origin master
