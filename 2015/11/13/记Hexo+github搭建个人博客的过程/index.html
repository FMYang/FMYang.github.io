记Hexo+github搭建个人博客的过程

搭建博客前的准备工作

1、安装Homebrew

Homebrew是Mac OSX上的软件包管理工具，能在Mac中方便的安装软件或者卸载软件，在终端执行下面命令安装：

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装完成后可以使用下面命令查看版本信息

npm -v

2、使用Homebrew安装node

在终端执行下面命令安装

brew install node

安装完成后可以使用下面命令查看版本信息

node -v

开始搭建个人博客

安装Hexo

npm install --verbose -g hexo-cli

初始化Hexo
进入到相应的文件夹目

hexo init

npm install

安装完成后可在目录下看到对于目录结构

.
├── _config.yml  #站点配置文件
├── package.json  #应用程序的信息
├── scaffolds  #模板文件夹，博文是根据模板生成的
├── source  #资源文件夹
|   ├── _posts  #生成的博客
└── themes  #下载的主题

现在我们执行下列命令来发布的本地服务器

hexo g #生成博客静态文件
hexo s #启动本地服务器

在浏览器中打开http://localhost:4000查看默认效果

连接github

建立Repository，建立与你用户名对应的仓库，仓库名必须为【your_user_name.github.io】，固定写法

配置_config.yml与github建立关联

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repository: git@github.com:FMYang/FMYang.github.io.git
  branch: master

执行配置命令

hexo d

大功告成！

快去使用http://username.github.io/访问你的个人博客试试吧。
