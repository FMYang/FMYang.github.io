首先安利一波SDWebImage，SDWebImage是一个异步加载图片的框架，以UIImageView分类的方式实现，深受开发者喜欢，github上排行第一的图片框架，如果你还没用过，赶紧试试看吧~

使用之前不妨先了解下SDImageCache缓存清理策略
现在打开SDImageCache.h头文件，从类名可以看出缓存是在这个类处理的

接口声明上方有这么段注释

/**
 * SDImageCache maintains a memory cache and an optional disk cache. Disk cache write operations are performed
 * asynchronous so it doesn’t add unnecessary latency to the UI.
 */

通过注释我们知道SDImageCache包含内存缓存和磁盘缓存，写磁盘缓存的操作是异步执行的，不会阻塞UI
